
 # Flask To-Do Application

A simple **To-Do List web application** built using **Flask**, **SQLite**, and **SQLAlchemy**.
The app allows users to **create, view, update, and delete tasks** through a web interface and also provides **REST API endpoints**.

---

## Features

* Add, update, and delete tasks
* SQLite database integration
* CRUD operations using Flask
* REST API endpoints returning JSON

---

## Tech Stack

* Python
* Flask
* SQLite
* SQLAlchemy
* HTML

---

## Run the Project

```bash
python app.py
```

Open in browser:

```
http://127.0.0.1:5000
```

---

## API Endpoints

| Method | Endpoint          | Description     |
| ------ | ----------------- | --------------- |
| GET    | `/api/tasks`      | Get all tasks   |
| POST   | `/api/tasks`      | Create new task |
| PUT    | `/api/tasks/<id>` | Update task     |
| DELETE | `/api/tasks/<id>` | Delete task     |

Installation

git clone <repo_link>
cd flaskpro
pip install -r requirements.txt
python app.py

The application is deployed using Render.
